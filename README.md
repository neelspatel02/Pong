A classic Pong game built in Python using the `turtle` module.  

This game supports **single-player** and **two-player** modes. In single-player mode, you can choose the difficulty level for the computer opponent: **Easy**, **Medium**, or **Hard**.

---

## Features

- Two game modes:
  - **Single-player**: Play against the computer with adjustable difficulty.
  - **Two-player**: Play against a friend on the same keyboard.
- Difficulty levels for single-player:
  - **Easy**: Slow computer paddle movement.
  - **Medium**: Moderate computer paddle movement.
  - **Hard**: Fast and challenging computer paddle movement.
- Simple and clean interface using Pythonâ€™s `turtle` module.
- Keyboard controls for paddle movement.

---

## Controls

- **Player 1 (Left Paddle)**: `W` (up), `S` (down)  
- **Player 2 (Right Paddle / or Computer)**: `Up Arrow` (up), `Down Arrow` (down)  

> In single-player mode, Player 1 is controlled by the computer.

---

## Requirements

- Python 3.x
- `turtle` module (comes with standard Python installation)
